Nhận diện khuôn mặt

Tổng quan​

Nhận diện khuôn mặt là giải pháp nhận diện khuôn mặt cực nhanh đi kèm với 6 điểm mốc và hỗ trợ đa khuôn mặt

Cách sử dụng​

main.py
from nkdcv.FaceDetectionModule import FaceDetector
import cv2

cap = cv2.VideoCapture(0)
detector = FaceDetector()

while True:
    success, img = cap.read()
    img, bboxs = detector.findFaces(img)

    if bboxs:
        # bboxInfo - "id","bbox","score","center"
        center = bboxs[0]["center"]
        cv2.circle(img, center, 5, (255, 0, 255), cv2.FILLED)

    cv2.imshow("Image", img)
    cv2.waitKey(1)

FaceDetector () sẽ gọi API tới FaceDetectionModule

Sau đó, sử dụng detector.findFaces (img) sẽ phát hiện vị trí của khuôn mặt và hiển thị một hộp xung quanh khuôn mặt
